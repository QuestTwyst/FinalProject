# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

User – any registered person who can both read stories and create/publish their own branching stories.

## Draft User Stories

1. As a user, I want to create an account, so that I have my information saved.
2. As a user, I want to log in to my account, so that I can access my saved stories and progress.
3. As a user, I want to browse a list of available stories, so that I can choose one to read.
4. As a user, I want to filter stories by genre, so that I can find a story that matches what I'm in the mood for.
5. As a user, I want to read a story one passage at a time, so that the plot unfolds gradually.
6. As a user, I want to choose between two options at the end of a passage, so that I can influence how the story unfolds.
7. As a user, I want to reach different endings based on my choices, so that I can experience multiple storylines within the same story.
8. As a user, I want to restart a story from the beginning, so that I can explore a different path and ending.
9. As a user, I want to create a new story, so that I can share my own branching narrative with others.
10. As a user, I want to add passages to my story, so that I can build out the plot.
11. As a user, I want to add two choices to each passage, so that readers can make decisions that branch the story.
12. As a user, I want to mark a passage as an ending, so that the story can conclude at that point.
13. As a user, I want to edit an existing passage in my story, so that I can fix mistakes or improve the writing.
14. As a user, I want to delete a passage from my story, so that I can remove content I no longer want included.
15. As a user, I want to view a list of stories I've created, so that I can manage and continue working on them.
